# Rhythm Box

Rhythm Box is a drum machine developed as a personal project using **Python**, **Pure Data**, and **Raspberry Pi**.

The project currently includes two versions:

## Raspberry Pi Version

* Python graphical interface
* Pure Data audio engine
* OSC communication between Python and Pure Data
* Hardware controls using potentiometers
* OLED display support
* Designed to run on a Raspberry Pi with a touchscreen

## PC Version

* Standalone desktop version
* Python graphical interface
* Pure Data audio engine
* Intended for development, testing, and use without Raspberry Pi hardware

## Features

* Step sequencer
* Sample-based drum machine
* Touchscreen interface
* Hardware control support
* OSC communication
* Real-time pattern editing

## Technologies

* Python
* Pure Data (Pd)
* Raspberry Pi
* Kivy
* OSC (Open Sound Control)
* MCP3008 ADC
* SSD1306 OLED Display

## Project Status

The project is fully functional and is still under active development. Future updates will include additional features, improvements, and more complete documentation.
